
### `docker-compose up`  

Runs the app.<br>
- [http://localhost:3000](http://localhost:3000) Client.<br>
- [http://localhost:4000](http://localhost:4000) Server.

## Packages I used 
Client :
 - **typescript**
 - **prop-types**
 - **node-sass**
 - **react-paginate**
 
Server :
 - **express**
 - **cors**
 - **express-async-await**
 - **node-fetch**
